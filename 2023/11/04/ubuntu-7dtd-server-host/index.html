1.设置swap

首先检查一下

如果没有，按如下步骤创建Swap文件。一般建议swap大小设置为自己物理内存相同或两倍。这里设置了4G。

启用swap文件

修改fstab让配置永久生效。

文件末尾加入

swappiness参数决定了系统将数据从内存交换到swap空间的频率。该数值越接近于0，系统越倾向于不进行swap，最大限度使用物理内存。数值越接近100，越倾向于使用swap，而过多的使用swap可能导致服务器运行效率缓慢。ubuntu系统的默认推荐值是60，为了保证服务器运行效率，建议设置为10。

端口 协议 说明
8080 TCP Web控制台
