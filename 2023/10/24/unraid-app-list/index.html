记录一下现在用的unread的插件及docker应用，基本都可以在app商店搜到

插件：

  • Appdata Backup（自动备份app数据）
  • CA Auto Update Applications（自动升级app）
  • CA Cleanup Appdata（清理app数据）
  • Community Applications（app商店）
  • Dynamix File Manager（系统文件管理）
  • Dynamix S3 Sleep（系统休眠工具）
  • Dynamix System Temperature（显示更多硬件温度数据）
  • Fix Common Problems（自动查找系统问题并提出解决方法）
  • GPU Statistics（GPU状态面板）
  • Intel GPU TOP（intelGPU驱动）
  • unBALANCE（迁移文件数据）
  • Unraid Connect
  • User Scripts（自定义脚本）

Docker app

  • adminer （其实可以不用，只是方便以图形UI管理数据库）
  • Apache-WebDAV（远程webdav服务）
  • aria2-pro（aria2下载工具）
  • jellyfin（多媒体服务器）
    * mariadb
    * PhotoPrism（照片库）
  • v2rayA（科学上网）
    * AutoBangumi（自动追番，通过estrellaxd/auto_bangumi安装）
  • qbittorrent（自动下番）
  • Heimdall（整理应用）
  • ddns-go
  • Komga（漫画阅读器）
    * TinyMediaManager（手动刮削器）
  • ZeroTier（没有公网IP就靠它了）
  • NginxProxyManager（反向代理）
  • FileBrowser（简单的文件浏览）
  • Krusader（文件管理，主要可以压缩解压）
  • transmission（挂PT）
  • syncthing（同步文件）
